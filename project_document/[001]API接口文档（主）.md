# 项目管理服务

## POST 创建项目（仅教师）

POST /api/project

> Body 请求参数

```json
{
  "projectName": "软件项目管理课程项目123123123",
  "description": "2024春季学期项目",
  "startTime": "2024-03-01T00:00:00",
  "endTime": "2024-06-30T23:59:59",
  "maxGroupSize": 6,
  "minGroupSize": 3,
  "groupDeadline": "2024-03-15T23:59:59",
  "allowStudentCreateGroup": 1
}
```

### 请求参数

|名称|位置|类型|必选|中文名|说明|
|---|---|---|---|---|---|
|Authorization|header|array[string]| 否 ||none|
|body|body|object| 否 ||none|
|» projectName|body|string| 是 | 项目名称|none|
|» description|body|string| 是 | 描述|none|
|» startTime|body|string| 是 | 开始时间|none|
|» endTime|body|string| 是 | 结束时间|none|
|» maxGroupSize|body|integer| 是 | 最大小组容量|none|
|» minGroupSize|body|integer| 是 | 最小小组容量|none|
|» groupDeadline|body|string| 是 | 小组创建截止时间|none|
|» allowStudentCreateGroup|body|integer| 是 | 是否允许学生创建小组（1-允许，0-不允许，默认1）|none|

> 返回示例

```json
{
  "code": 200,
  "message": "操作成功",
  "data": {
    "id": null,
    "projectId": "61954be7-401a-4f43-9ae4-911055ca88b6",
    "projectName": "软件项目管理课程项目",
    "projectCode": "PRJ20260213BD8",
    "description": "2024春季学期项目",
    "creatorId": "ADMIN_USER_001",
    "startTime": "2024-03-01T00:00:00",
    "endTime": "2024-06-30T23:59:59",
    "status": 1,
    "maxGroupSize": 6,
    "minGroupSize": 3,
    "groupDeadline": "2024-03-15T23:59:59",
    "allowStudentCreateGroup": 1,
    "createTime": null,
    "updateTime": null
  }
}
```
